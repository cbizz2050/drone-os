# Parts list for Autonomous drone

ESCs: Simonk 30A  (x4)

Props: 1045/1045R

    Power Requirement: 5V DC 2A

Camera: Luxonis OAK-D
    Power Requirement: 5V V DC 1.5A - 2A

Power Supply: Pi Sugar2 5v 5000mAh
